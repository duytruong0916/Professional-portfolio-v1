AnguBetta Store
An online expense tracking app.
AnguBetta is an E-commerce web app. The application worked as a Betta fish store in which users can explore an amazing world of Betta fish and pick their favorite one for a reasonable price.

This application is 60% completed. Some front-end and back-end implementation still needs to be done, including fetching data from the server, user's shopping cart and orders, user checkout and so on.
